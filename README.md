# BPMN 编辑器

符合「中文使用习惯」和「标准规范」的 BPMN 编辑器<br/>
- 对中文进行深度优化
- 基于 bpmn-js 遵循 BPMN.io 规范

## 特性

- 界面汉化
- 字体优化（中文优先）
- 字体大小优化（更适合中文内容的阅读）
- 界面元素优化（更符合中文信息密度高的特点）
