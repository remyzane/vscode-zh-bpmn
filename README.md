# BPMN 编辑器

[![Visual Studio Marketplace Version](https://img.shields.io/visual-studio-marketplace/v/chinese.vscode-zh-bpmn?logo=visual-studio-code&label=VS%20Code%20插件)](https://marketplace.visualstudio.com/items?itemName=chinese.vscode-zh-bpmn)

**符合「中文使用习惯」和「标准规范」的 BPMN 编辑器**<br/>
- 根据中文进行深度优化
- 遵循 BPMN 规范（基于 bpmn-js）

## 中文优化

- 界面汉化
- 字体优化（中文优先）
- 字体大小优化（更适合中文内容的阅读）
- 界面元素优化（更符合中文信息密度高的特点）
